StepZen - Frankfurter GraphQL API

Frankfurter is an open-source API for current and historical foreign exchange rates published by the European Central Bank. Using GraphQL! View GraphiQL playground

Name Description

Query
query frankfurter_historical_rates($from: String, $to: String, $amount: Float, $date: Date) {
  frankfurter_historical_rates(from: $from, to: $to, amount: $amount, date: $date) {

frankfurter_time_series

Returns a Frankfurter_TimeSeries_Rates

Name Description
start_date - Date
end_date - Date
from - String
to - String
amount - Float

Example

Query
query frankfurter_time_series($start_date: Date, $end_date: Date, $from: String, $to: String, $amount: Float) {
  frankfurter_time_series(start_date: $start_date, end_date: $end_date, from: $from, to: $to, amount: $amount) {
    amount
    base
    end_date
    rates
    start_date
  }
}
Variables
{
  "start_date": "2022-02-28T18:52:04.796Z",
  "end_date": "2022-02-28T18:52:04.796Z",
  "from": "xyz789",
  "to": "xyz789",
  "amount": 987.65
}
Response 
{
  "data": {
    "frankfurter_time_series": {
      "amount": 987.65,
      "base": "abc123",
      "end_date": "2021-08-28T17:52:04.796Z",
      "rates": {},
      "start_date": "2021-08-28T17:52:04.796Z"
    }
  }
}
